# runtime-tester

<p>
	This projects provides simple means to test the running time of an algorithm. All you need to do is to put the code you want ot test into a Runnable and give it to the <b>measure</b> function of the <b>RuntimeTester</b> class.
</p>
<p>
	The function <b>measure</b> has overloed versions. You can also test Consumer<T> like of Function<T,R> like functions.
</p>
<p>
	By the measureEach function, you can also get every test results instead of ana overall average. 
</p>
